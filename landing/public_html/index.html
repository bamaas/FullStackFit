About

A gym tracker to monitor your massive gains. This project is started from a interest in learning to program and a lack of suitable app for my personal need. This project is fully open source and it's running on a personal server at my own home. There are no ads, no fees, but also no guarantees on availability.

Open Source

This project is fully open source and thus all the source code is visible at Github. It makes exclusive use of open source technologies, such as; Postgres, Angular, Spring Boot, Keycloak, Proxmox, Kubernetes, Rancher, Cypress...

Respected privacy

Your data is yours and will never be shared with a third party. All the data is stored on my own server at home. Off-shore backups are made to an Wasabi S3 bucket.

Mobile first

This app is built with a mobile first aproach. It should work flawlessly on every device, from mobile to desktop.
